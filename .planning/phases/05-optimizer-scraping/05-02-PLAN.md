  <action>
    **Per D-11, D-12, D-13:**

    1. **services.py** — Two functions:

       `extract_text_from_image(image_bytes: bytes, lang: str = "spa+eng") -> list[str]`:
       - Open image with `PIL.Image.open(io.BytesIO(image_bytes)).convert("L")` (grayscale)
       - Apply `ImageOps.autocontrast(image)` and `image.filter(ImageFilter.SHARPEN)`
       - Legacy approach: run `pytesseract.image_to_string(image, lang=lang, config="--psm 6 --oem 3")`
       - Split result by newlines, strip whitespace, filter out empty lines
       - If no text extracted, raise `OCRProcessingError("No se pudo extraer texto de la imagen")`
       - Return list of non-empty stripped lines

       `match_products(raw_lines: list[str], threshold: int = 80) -> list[dict]`:
       - Query `Product.objects.filter(is_active=True).values("id", "name")`
       - For each line in raw_lines:
         - Compare against all products using `thefuzz.fuzz.token_sort_ratio(line.upper(), product["name"].upper())`
         - Track best_score and best_product
         - Build result dict: `{"raw_text": line, "confidence": best_score / 100.0, "quantity": 1}`
         - If `best_score >= threshold` and best_product: add `"matched_product_id": best_product["id"]` and `"matched_product_name": best_product["name"]`
       - Return list of result dicts

    2. **serializers.py** — Three serializers:

       `OCRScanRequestSerializer(serializers.Serializer)`:
       - `image = serializers.ImageField(required=True, help_text="Foto de lista o ticket")`

       `OCRItemSerializer(serializers.Serializer)`:
       - `raw_text = serializers.CharField()`
       - `matched_product_id = serializers.IntegerField(required=False, allow_null=True)`
       - `matched_product_name = serializers.CharField(required=False, allow_null=True)`
       - `confidence = serializers.FloatField()`
       - `quantity = serializers.IntegerField(default=1)`

       `OCRScanResponseSerializer(serializers.Serializer)`:
       - `items = OCRItemSerializer(many=True)`

    3. **views.py** — `OCRScanView(APIView)`:
       - `permission_classes = [IsAuthenticated]`
       - `parser_classes = [MultiPartParser]`
       - `throttle_scope = "ocr"`
       - `post(self, request)`:
         - Validate request with `OCRScanRequestSerializer(data=request.data)`
         - Read image bytes: `request.FILES["image"].read()`
         - Call `extract_text_from_image(image_bytes)`
         - Call `match_products(raw_lines)`
         - Serialize with `OCRScanResponseSerializer({"items": results})`
         - Return `Response({"success": True, "data": serializer.data})`
         - Catch `OCRProcessingError` and return 422 with error format
         - Catch `Exception` broadly and return 500 with generic message, log with structlog

    4. **urls.py** — Replace empty urlpatterns:
       ```python
       from django.urls import path
       from .views import OCRScanView

       urlpatterns = [
           path("scan/", OCRScanView.as_view(), name="ocr-scan"),
       ]
       ```
