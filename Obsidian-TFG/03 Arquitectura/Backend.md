# Backend

## Resumen

El backend Django organiza el dominio por apps y expone la API REST consumida por móvil y web.

## Estado

Operativo en el bloque core.

## Puntos clave

- Apps principales ya implementadas.
- Tests unitarios e integración presentes.
- Documentación API disponible.
- Optimizer/OCR/assistant/scraping quedan como bloque de evolución.
